Return per-day or per-month scan counts, broken down by terminal
status. Useful for charting throughput over time.

```http
GET /api/v2/org/usage/scans
```

## Query parameters

<ParamField query="range" type="string" default="daily">
  `daily` returns the last 30 days. `monthly` returns the last 12
  months.
</ParamField>

## Response

<ResponseField name="range" type="string" required>
  Echoes the `range` you requested.
</ResponseField>

<ResponseField name="points" type="object[]" required>
  One entry per day or month. Missing periods are filled with zeros so
  the series is continuous.
</ResponseField>

<ResponseField name="updated_at" type="string" required>
  ISO 8601 UTC timestamp of when the data was generated.
</ResponseField>

<ResponseField name="source" type="string" required>
  `cache` if served from the warmer cache, `live` if computed on the
  fly.
</ResponseField>

### `points[]`

<ResponseField name="date" type="string" required>
  ISO 8601 UTC timestamp pinned to `T00:00:00Z`. For monthly ranges,
  always the first of the month.
</ResponseField>

<ResponseField name="total" type="integer" required>
  Scans submitted in this period (sum of the status-broken-out
  fields).
</ResponseField>

<ResponseField name="completed" type="integer" required>
  Scans whose final status was `completed`.
</ResponseField>

<ResponseField name="completed_with_partial" type="integer" required>
  Scans whose final status was `completed_with_partial`.
</ResponseField>

<ResponseField name="failed" type="integer" required>
  Scans whose final status was `failed`.
</ResponseField>

<ResponseField name="processing" type="integer" required>
  Scans still in flight at the time the response was generated.
</ResponseField>

## Example

```bash
curl "https://api.tumban.com/api/v2/org/usage/scans?range=daily" \
  -H "Authorization: Bearer sk_xxx"
```

```json
{
  "range": "daily",
  "points": [
    {
      "date": "2026-04-29T00:00:00Z",
      "total": 42,
      "completed": 40,
      "completed_with_partial": 1,
      "failed": 1,
      "processing": 0
    }
  ],
  "updated_at": "2026-04-29T12:00:00Z",
  "source": "cache"
}
```
